### core-image-xhr

一个简单的图片上传库

### Start

``` bash 
npm i core-image-xhr --save
```

``` js
import xhr from 'core-image-xhr';

xhr(method, url, header, data, successCallback, errorCallback);
```


+ method 'POST' or 'GET'

+ url

+ header Object

+ data Object

+ successCallback function

+ errorCallback function
